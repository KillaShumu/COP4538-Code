📊 Data Structures Used:

  • Category Tree: Tree structure for hierarchical contact organization (Work -> Department -> Team)
  • BST for Categories: Backend category index for faster category retrieval (not directly wired to the UI)
  • VIP Priority Queue: Heap-based priority queue ensuring high-priority contacts appear at top
  • Undo: Stack (LIFO) — stores add/delete operations
  • Redo: Queue (FIFO) — stores undone operations
